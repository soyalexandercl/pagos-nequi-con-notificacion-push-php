Este es un proyecto básico en PHP para realizar cobros mediante notificaciones de Nequi.
El sistema funciona con tres archivos que manejan todo el proceso: uno se encarga de generar el token, otro envía la solicitud de pago al teléfono del cliente y el tercero verifica el estado del cobro. Simplemente configuras tus credenciales y el código hace el resto.
Para obtener los datos de Nequi deben de contactarse por medio de Conecta Nequi.
